# GuessingGame
A simply guessing game, the goal is to find the targetNumber between 0 and 100.


Requirements
------------

  * NPM;
  * Node js; 
 

Installation
------------

### Step 1 : Clone the project

Now clone the project from github.

```bash
$ git clone https://github.com/MGardier/GuessingGame.git
```

### Step 2 : Install dependencies

Now that the project is cloned, running the following command should install all the npm dependencies:

```bash
$ npm install
```

### Step 3: Run the project

Now run this command in the terminal:

```bash
$ node index.js
```

## **And That's it.**
